# Project Statement

## 1. Problem Statement
Students and professionals often struggle to manage their daily tasks effectively, leading to missed deadlines and lower productivity. Existing tools are either too complex or lack simple analytics to track performance.

## 2. Scope of the Project
This project is a "Task & Productivity Planner" backend system. It allows users to register, manage tasks with priorities, and view analytics on their completion rates.

## 3. Target Users
* **Students:** To track study schedules and assignments.
* **Freelancers/Professionals:** To manage project deadlines.

## 4. High-Level Features
* **User Management:** Secure registration and login (JWT Authentication).
* **Task Management:** CRUD operations for tasks (Create, Read, Update, Delete).
* **Analytics:** Calculation of productivity scores and pending task summaries.
